## Building

#### NPM
Clone this repo
````bash 
git clone https://github.com/ola-wale/Mobile-Applications-Project--Portfolio-App.git
````
Install Dependencies
````bash 
npm i
````
Serve with ionic
````bash 
ionic serve
````
